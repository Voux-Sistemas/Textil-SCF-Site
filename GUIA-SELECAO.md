# Têxtil SCF - Guia de Seleção do Site

**Para: Thaís e família.**

Este guia é um cardápio: vocês marcam o que querem que o site tenha, respondem
as perguntinhas de cada parte escolhida, e devolvem pra gente. Com isso a gente
escreve os textos finais e monta o site do jeito de vocês.

**Como usar:**
1. Abram o site de demonstração enquanto leem:
   **https://textil-scf-site-production.up.railway.app**
   (Tudo ali é exemplo: textos, números e fotos serão trocados pelos reais.)
2. Marquem ✅ no que querem e ❌ no que não querem.
3. Onde tiver pergunta, respondam do lado. Pode ser à mão numa folha, foto,
   áudio de WhatsApp, como for mais fácil.
4. Na dúvida, sigam a opção marcada como **(nossa sugestão)**.

---

## PARTE 1 - O tom (como o site "fala")

A mesma empresa pode soar de jeitos diferentes. Vejam a frase de abertura
escrita de três formas e escolham a que soa como vocês:

**( ) A - Técnico e preciso**
> "Estamparia digital de alta definição. Cor fiel ao padrão, da arte à fixação."
Passa: precisão, indústria, confiança técnica.

**( ) B - Tradição e família**
> "Há [30] anos, a nossa família coloca cor no tecido das marcas brasileiras."
Passa: história, proximidade, casa de confiança.

**( ) C - Editorial, de moda** *(é o que está no site hoje)*
> "A cor que o tecido não esquece."
Passa: sofisticação, marca premium, estilo.

Pode misturar (ex.: "C nas frases grandes, A nos textos técnicos"). Anotem:
_______________________________________________

**Pergunta de tom:**
- O site deve dizer que é uma **empresa familiar**? ( ) Sim ( ) Não

## PARTE 2 - As seções (marquem o que querem)

### 1. Abertura ✦ fixa
A primeira tela: frase grande + dois botões (Ver o acervo / Área do Cliente).
Todo site tem. A frase vem do tom escolhido na Parte 1.
- **Precisamos:** ano de fundação: ________

### 2. Manifesto - a frase de posicionamento
Um bloco de destaque com a "tese" da casa, tipo:
> "A estampa é a primeira coisa que a cliente vê. Nós a tratamos como a assinatura da marca."
( ) ✅ Quero  ( ) ❌ Não quero  **(nossa sugestão: ter)**
- **Precisamos (a pergunta mais importante do guia):**
  Por que um cliente escolhe VOCÊS e não o concorrente? Respondam com suas
  palavras, sem filtro: ________________________________________________

### 3. Números - a faixa de credibilidade
Uma faixa com 2 ou 3 números fortes. Marquem quais usar (e escrevam o valor real):
- ( ) Anos de mercado: ______
- ( ) Resolução de impressão (dpi): ______
- ( ) Metros estampados por mês: ______
- ( ) Estampas no acervo: ______
- ( ) Outro: ______________________
( ) ✅ Quero  ( ) ❌ Não quero  **(nossa sugestão: ter, com 2 números)**

### 4. Foto panorâmica da fábrica
Uma faixa larga com foto do galpão/produção, de respiro entre as seções.
(No site de demonstração tem uma imagem de exemplo gerada; a de vocês entra no lugar.)
( ) ✅ Quero  ( ) ❌ Não quero  **(nossa sugestão: ter)**
- **Precisamos:** 1 foto boa e ampla da fábrica (ou combinamos de produzir).

### 5. Serviços - o que a casa faz
Cards com os serviços. Hoje o site mostra estes 4. Corrijam a lista:
- ( ) Impressão digital (metragem corrida)
- ( ) Sublimação
- ( ) Estampa localizada
- ( ) Desenvolvimento de estampa exclusiva
- ( ) Falta algum? ______________________
- Atendem **baixa metragem / pedido mínimo pequeno**? ( ) Sim, e é um diferencial a destacar ( ) Sim, mas sem destacar ( ) Não
( ) ✅ Quero  ( ) ❌ Não quero  **(nossa sugestão: ter)**

### 6. Acervo - a galeria de estampas
As estampas aparecem **desfocadas de propósito**, com marca d'água e cadeado:
quem quiser ver nítido entra na Área do Cliente. Isso protege o acervo de cópia
e ainda desperta curiosidade.
Escolhas:
- Quantas estampas na vitrine? ( ) 6  ( ) 9  ( ) 12 **(sugestão: 6)**
- Filtros por família (Botânica, Geométrica, etc.)? ( ) Sim ( ) Não
- Quais famílias vocês mais produzem? ______________________
( ) ✅ Quero  ( ) ❌ Não quero  **(nossa sugestão: ter, é o coração do site)**
- **Precisamos:** fotos das estampas escolhidas (a gente orienta como fotografar).

### 7. Processo - o passo a passo da estampa
Mostra as etapas do trabalho, com fotos das máquinas. Hoje o site mostra 5 passos:
> 01 Arte · 02 Preparação · 03 Impressão · 04 Fixação · 05 Controle de cor
Confirmem ou corrijam: quais são as etapas de verdade, na ordem, com o nome que
vocês usam no dia a dia? (Pode ser 4, pode ser 6.)
1. ________  2. ________  3. ________  4. ________  5. ________  6. ________
( ) ✅ Quero  ( ) ❌ Não quero  **(nossa sugestão: ter, dá muita autoridade)**

### 8. A Casa - a história
A parte mais pessoal do site. Três formatos possíveis:
- ( ) **A - Completa, com linha do tempo** (texto + marcos com ano) **(nossa sugestão)**
- ( ) **B - Só um texto curto**, sem linha do tempo
- ( ) **C - Não ter seção de história**
Se escolherem A ou B, **precisamos:**
- Como a empresa começou? Quem fundou, por quê? ______________________
- 3 a 5 marcos com ano (ex.: fundação, primeira impressora digital, estúdio
  próprio de criação, mudança de sede, certificação):
  ______________________
- Certificações ou selos que têm (ABVTEX, etc.): ______________________

### 9. Rodapé / Contato ✦ fixo
Todo site tem. Marquem os canais que devem aparecer (e escrevam o dado):
- ( ) WhatsApp: ______________________
- ( ) E-mail: ______________________
- ( ) Instagram: @____________________
- ( ) Endereço: ______________________
- ( ) Horário de atendimento: ______________________

## PARTE 2B - Seções extras (não existem hoje, mas dá pra criar)

- ( ) **Sustentabilidade / certificações** - só vale se tiverem ações ou selos
  concretos (senão soa vazio).
- ( ) **As pessoas da casa** - fotos e nomes da equipe/família. Humaniza bastante.
- ( ) **Perguntas frequentes** - pedido mínimo, prazo, como enviar a arte.
  Bom pra reduzir perguntas repetidas no WhatsApp.
- ( ) **Marcas que atendemos** - atenção: só com autorização das marcas.
  Em geral não recomendamos; a discrição também é um valor no B2B.

## PARTE 4 - Como devolver

Mandem de volta neste formato (foto da folha preenchida ou mensagem):

```
TOM: (A, B, C ou mistura)
EMPRESA FAMILIAR NO SITE: sim/não
SEÇÕES: (as que marcaram ✅)
NÃO QUEREMOS: (as que marcaram ❌)
RESPOSTAS: (as respostas das perguntas de cada seção)
FOTOS: (o que já têm de foto e o que precisaria produzir)
```

Com isso a gente escreve todos os textos no tom escolhido e devolve o site
pronto pra aprovação. Qualquer dúvida durante a reunião, é só chamar.
